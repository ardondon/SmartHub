## 安装与配置

### Q1: SmartHub 支持哪些 Android 版本？

**A:** SmartHub 支持 Android 8.0 (API 26) 及以上版本。推荐使用 Android 10 或更高版本以获得最佳体验。

### Q2: 可以在手机上使用吗？

**A:** 可以，但我们更推荐在平板上使用：
- **平板**：8-10寸大屏，完美的控制中心体验 ⭐推荐
- **手机**：功能完整，但界面布局针对小屏优化

### Q3: 如何安装 SmartHub？

**A:** 三种方式：

1. **下载 APK（推荐）**
   - 访问 [Releases](https://github.com/ardondon/SmartHub/releases)
   - 下载最新 `.apk` 文件
   - 在设备上启用"未知来源"
   - 安装 APK

2. **Google Play**（即将上线）
   - 搜索 "SmartHub"
   - 点击安装

3. **从源码编译**
   - 参考 [开发指南](../CONTRIBUTING.md)

### Q4: 首次配置需要注意什么？

**A:** 按以下步骤操作：

1. ✅ 连接 WiFi（必须）
2. ✅ 添加云平台账号
3. ✅ 同步设备
4. ✅ 配置场景（可选）

⚠️ 注意：确保平板和智能设备在同一 WiFi 网络下。

### Q9: 如何添加云平台账号？

**A:** 

1. 打开 SmartHub
2. 进入"设置" → "云平台管理"
3. 点击"添加平台"
4. 选择平台（如：腾讯连连）
5. 输入账号密码登录
6. 授权 SmartHub 访问设备

### Q10: 为什么登录云平台失败？

**A:** 可能的原因：

1. **账号密码错误** - 重新输入
2. **网络问题** - 检查网络连接
3. **平台维护** - 稍后再试
4. **需要二次验证** - 在原生 App 中授权
5. **区域限制** - 部分平台仅支持特定区域

### Q11: 云平台 API 变更会影响使用吗？

**A:** 我们会及时适配：

- ✅ 插件架构隔离，影响范围小
- ✅ 自动更新插件
- ✅ 降级保护，确保核心功能可用

### Q13: 二手平板有什么需要注意的？

**A:** 

1. **屏幕** - 无坏点、无黄斑
2. **电池** - 健康度 >80%
3. **网络** - WiFi/蓝牙功能正常
4. **系统** - 能升级到 Android 8.0+
5. **来源** - 闲鱼/拼多多，选择信誉商家

### Q14: 平板需要一直充电吗？

**A:** 

- **推荐做法**：挂墙使用，常驻充电
- **电池保护**：SmartHub 内置充电优化
  - 充满后自动断电
  - 电量 <20% 时提醒
- **长期不用**：充电至 50%，关机存放

### Q15: 支持多个平板吗？

**A:** 

- **免费版**：单个网关
- **专业版**：支持多网关协同
  - 数据自动同步
  - 负载均衡
  - 故障切换

## 故障排除

### Q16: 应用闪退怎么办？

**A:** 

1. **清除缓存**
   - 设置 → 应用 → SmartHub → 清除缓存
   
2. **重启应用**
   - 强制停止后重新打开
   
3. **重装应用**
   - 卸载后重新安装
   
4. **检查日志**
   - 设置 → 关于 → 导出日志
   - 提交到 [Issues](https://github.com/ardondon/SmartHub/issues)

### Q17: 设备离线怎么办？

**A:** 

1. **检查设备**
   - 设备是否通电
   - WiFi 连接是否正常
   
2. **检查网关**
   - SmartHub 是否在线
   - 平板网络是否正常
   
3. **重新绑定**
   - 删除设备
   - 重新添加

### Q18: MQTT 连接失败？

**A:** 

1. **检查网络**
   - ping 测试
   - 防火墙设置
   
2. **检查配置**
   - Broker 地址
   - 端口号（默认 1883）
   - 认证信息
   
3. **查看日志**
   - 设置 → 日志 → MQTT

### Q19: 性能卡顿怎么办？

**A:** 

1. **清理内存**
   - 关闭后台应用
   
2. **减少设备数量**
   - 免费版限制 10 台
   - 升级订阅
   
3. **优化场景**
   - 减少复杂条件
   - 合并相似场景
   
4. **更换设备**
   - 推荐 4GB+ RAM

### Q20: 如何导出配置？

**A:** 

1. 打开"设置"
2. "备份与恢复"
3. "导出配置"
4. 保存到本地

**恢复配置：**
1. "导入配置"
2. 选择备份文件
3. 确认恢复
